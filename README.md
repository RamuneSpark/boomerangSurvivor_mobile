## ゲームの始め方

ページを開くと、『ブーメランサバイバー モバイル』のタイトルロゴとともにスタート画面が表示されます。<br>各種設定した後に、スタート画面の「ゲームスタート!」をタップするとゲームが始まります。


### 設定項目

スタート画面では、下記項目を変更することができます。

* **タイム設定**
  - 「2分モード」か「5分モード」のどちらかを選ぶことができます。
    
* **プラクティス設定**
  - 「スタンダード」か「プラクティス」のどちらかを選ぶことができます。
  - 「プラクティス」は敵の攻撃がやさしくなります。初めての方や少しずつゲームに慣れたいという方におすすめです。
  - ハイスコアを狙う場合は「スタンダード」がおすすめです。
    
* **サウンド設定**
  - サウンドの有無を選択できます。無効にすると、すべての音声が再生されなくなります。
    
* **BGM設定**
  - サウンド設定を有効にすると変更できます。
  - タップするごとに「BGM 1」→「BGM 2」→「BGM 3」→「BGM OFF（効果音のみ）」と切り替えることができ、併せてゲームプレイ中のBGMが変化します。
  - ボタンを長めにタップすると逆の順番で切り替わります。



## ゲームの遊び方

設定したタイムが0になるか、敵に触れてミスになってしまうまでにどれだけのスコアが獲得できるかに挑戦します。

スコアは、敵を倒したりアイテムを獲得したりするだけではなく、特定の条件によって終了時にボーナスとして加算されることもあります。<br>ハイスコアを目指しましょう！

**メイプルの操作方法については、スタート画面の「あそびかた」からご確認ください。**


### アイテム

* **スピードアップ**
  - 移動速度レベルが1上がります（MAX 8）。
 
* **アタックアップ**
  - ブーメランの飛距離・速度レベルが1上がります（MAX 8）。

* **ハート**
  - ミスをしても、このアイテムと引き換えにゲームを続行することができます。ただし、一度に持てるのは一つまでです。
 
* **ビッグウェポン**
  - ブーメランが少し大きくなり、敵に当たりやすくなります。
 
* **ボーナス**
  - 特定の条件を満たすと登場。高得点のチャンスです。


 
### 敵キャラクター紹介

* **ウサラ**
  - 森に暮らすウサギのような一頭身の生き物で、ならず者の個体が登場します。横に動くだけで、倒すのも簡単です。
 
* **お花さん**
  - 不気味の一つ目の花です。前方から現れ、まっすぐに迫ってきます。

* **スドーリ**
  - 森の中を駆ける鳥です。スピードに注意。
 
* **メタ**
  - 宇宙から姿を現した偵察ロボットです。回り込んで、メイプルの真後ろから突進を仕掛けてきます。
 
* **ツインクル**
  - メイプルに向かって一直線に飛んでくる流れ星で、**ブーメランで倒すことができない強敵**です。ひたすら逃げるのみ！


## ポーズ

ゲーム中に画面左上のボタンをタップすると、ゲームの進行を一時停止（ポーズ）することができます。

※ポーズ画面にゲームを途中で終了する機能はありません。途中でゲームを終了したい場合は、タブを閉じるか再読込をしてください。


## ブラウザバックと再読込について

このゲームは、ブラウザバックの操作によっていつでもページを離脱することができます。<br>また、再読込の操作によっていつでもゲームをリセットすることが可能です。<br>ただし、その際にゲームの進行状況などが消失した場合は、**元に戻すことはできません**のでご注意ください。<br><br>※このゲーム内でのキャンセル操作としてブラウザバックを使用することはできません。


## リザルト画面

ゲームが終了すると、リザルト画面が表示され、最終スコアが表示されます。<br>
同じ条件で再挑戦する場合は「YES」、条件を変更するかプレイを終了する場合は「NO」を選択してください。


### シェアボタン

各SNSに記録をシェアしたり、投稿内容をコピーすることができます。


### 記録カードのダウンロード

ゲームが終了すると、「**記録カード**」（JPG画像）をダウンロードできます。<br>記録カードには、スコアやゲームの条件、プレイタイム、ユーザーネームが表記され、記録を見返せる形で保管できます。<br><br>※記録カードの内容をこのゲームが外部に公開することはありません。<br>※ダウンロードまでには少し時間がかかることがあります。その間に**ダウンロードボタンを何度もタップすると、ダウンロードが正常にできなくなる可能性があります**ので、ご注意ください。<br>※15字以上のユーザーネームは使用できません。<br>※一度決めたユーザーネームは、次のゲームを終了するまで変更できません。<br>※発行した記録カードからハイスコアを読み込む（再現する）機能はありません。


### オンラインランキングへの登録

ゲームが終了すると、「**オンラインランキング**に登録」のボタンを押すことで、オンラインランキングにデータを登録できます。<br>オンラインランキングには、スコアやゲームの条件、プレイタイム、ユーザーネームが登録され、サーバーに記録されます。<br><br>※15字以上のユーザーネームは使用できません。<br>※一度決めたユーザーネームは、次のゲームを終了するまで変更できません。<br>※登録ボタンをタップしなかった場合、その回のスコアはオンラインランキングに登録されません。<br>上記「オンラインランキング規程」も、よくお読みください。


### 中断機能

このゲームでは、「**リザルト画面で最終スコアが表示されている状態**」に限り、中断データを一時的に保存します。

これにより、シェア機能等でページを離れて勝手に再読込が発生した場合も、リザルト画面から再開することができます。<br>
複数のSNSにシェアしたい方や、記録カードやオンラインランキングを後でご利用の方も、画面を安心して離れることができます。

なお、リザルト画面の中断データはプレイ終了時点から1時間以上経過すると無効となります。


## ランキング

スタート画面の「ランキング」にて、過去の記録を一部見ることができます。<br>「2分モード」と「5分モード」の切り替えも可能です。

オンラインランキングでは、「総合」と「月替わり」で表示を切り替えることもできます。

* **ローカル**
  - 過去の自己ベストを見返すことができます。こちらでは、事前にいくつかのプリセット記録が入っています（従来の「ハイスコア」画面です）。
 
* **トップ**
  - オンラインで、トップ5の記録が見られます。

* **ライバル**
  - オンラインで、自分の最高記録 + 前後およそ5つ分の記録が見られます。
 
* **ラスト**
  - オンラインで、自分の直前のプレイ記録 + 前後およそ5つ分の記録が見られます。


## セーブデータの保存について

このゲームでは、下記のデータを端末のローカルストレージにて保存します（オートセーブ）。

 * ローカルランキング
 * 最後に記録カードを作成した時のユーザーネーム（次回以降入力省略）
 * リザルト画面で中断したときの一時データ
 * 現在のBGM設定
 * （オンラインランキング登録者のみ）プレイヤーID

スタート画面「ランキング」→「ローカル」 → 右上の「CLEAR」ボタンをタップすることで、ローカルランキングを始めとするすべてのセーブデータを消去することができます。<br><br>※一度消去されたローカルランキングやプレイヤーIDを**元に戻すことはできません**ので、ご注意ください。<br>※ブラウザの仕様や設定によりローカルストレージの内容が消えてしまう可能性もありますので、予めご了承ください。
